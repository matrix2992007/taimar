# 🚀 Youssef Elite 3D Fitness App
### تطبيق رياضي تفاعلي بذكاء اصطناعي ومساعد صوتي

تطبيق ويب متكامل (PWA) مصمم خصيصاً للمبرمجين والرياضيين، يجمع بين واجهة المستخدم العصرية (Glassmorphism) وتقنيات التفاعل الصوتي لتحفيز المستخدم وقياس أدائه الرياضي بدقة.

---

## ✨ المميزات الرئيسية (Features)

* **Interactive 3D Mannequin:** هيكل بشري تفاعلي يتيح لك اختيار العضلة المستهدفة بمجرد اللمس.
* **AI Voice Assistant:** مساعد صوتي مدمج ينطق اسمك (يوسف) ويقدم لك تعليمات التمرين والتحفيز لحظياً.
* **PWA Ready:** يدعم التثبيت كـ تطبيق Desktop أو Mobile ويعمل بكفاءة **بدون إنترنت (Offline Mode)**.
* **Smart Health Analysis:** تحليل فوري لمؤشر كتلة الجسم (BMI) وتحديد هدفك تلقائياً (تضخيم/تنشيف) مع نصائح غذائية.
* **Performance Tracking:** نظام ذكي لمراقبة تطورك باستخدام الأسهم التفاعلية (🔼/🔽) لمقارنة أدائك الحالي مع أرقامك القياسية.
* **Two Training Modes:** التبديل بين نظام "الثواني" و نظام "العدّات" (كل ثانيتين عدة) لضمان التكنيك الصحيح.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5 & Semantic Tags:** لبناء هيكل متوافق مع معايير الويب.
* **CSS3 (Glassmorphism & Flex/Grid):** لتصميم واجهة نيون عصرية مريحة للعين.
* **Modern JavaScript (ES6+):** لإدارة المنطق البرمجي والعمليات الحسابية.
* **Web Speech API:** لتمكين المساعد الصوتي من التحدث باللغة العربية.
* **Service Workers & Manifest:** لتحويل الموقع إلى تطبيق يعمل أوفلاين.
